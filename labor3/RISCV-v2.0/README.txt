Projeto do Processador RISC-V v2.0

Para montar o projeto:
1) Abra o Quartus Prime v.18.1
2) Escolha Project/Restore Archieved Project
3) Escolha o arquivo Core.qar
4) Escolha o diretório de destino C:/.../RISCV-v2.0/Core   (sem o _restored)
5) Pronto! :)

No diretório Sofs estão os processadores já sintetizados.

Lembre-se que no início do Parametros.v:
-Escolha o Processador UNICICLO, MULTICICLO ou PIPELINE
-Escolha a ISA RV32I, RV32IM ou RV32IMF

Para simulação em forma de onda use os arquivos Waveform1.vwf (mais curto) ou Waveform2.vwf (mais longo).
